Detailed Description

The MainWindow class inherits from QMainWindow and represents the main window of the application.

The MainWindow class is forward declared here. It is defined in the Ui namespace.

Parameters
parentis a pointer to the widget that is logically the parent of this window. It is passed to the QMainWindow constructor.
