Detailed Description

The VRRenderThread class inherits from the Qt class QThread which allows it to be a parallel thread to the main() thread, and also from vtkCommand which allows it to act as a "callback" for the vtkRenderWindowInteractor. This callback functionality means that once the renderWindowInteractor takes control of this thread to enable VR, it can callback to a function in the class to check to see if the user has requested any changes.
