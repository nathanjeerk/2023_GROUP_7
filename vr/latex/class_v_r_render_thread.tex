The \doxylink{class_v_r_render_thread}{VRRender\+Thread} class inherits from the Qt class QThread which allows it to be a parallel thread to the main() thread, and also from vtk\+Command which allows it to act as a "{}callback"{} for the vtk\+Render\+Window\+Interactor. This callback functionality means that once the render\+Window\+Interactor takes control of this thread to enable VR, it can callback to a function in the class to check to see if the user has requested any changes.
